Lets see how we can drive global environment values from external files to tests in java

Create a file with .properties extensions , example data.properties

say we have browser , url of application to be driven from this properties file

properties file should look as below

        browser=chrome
        url=https://demowebsite.com

Step1: Create a java file

We have Properties class in Java under util package, define object of that method

Step2:Read the Properties file

To read file use FileInputStream , create object of it and pass .properties file path as a parameter to it

Step3:Load the file using object of Properties class and now we call the properties from .properties file using getProperty

Now lets look into code

        public class GlobalValuesDrive {
          public static void main(String[] args) throws IOException {
            //step1
            Properties prop = new Properties();
            //step2
            FileInputStream fis = new FileInputStream(
                "C:\\Testing\\SeleniumJava\\FrameworkBasics\\TestNGTutorial\\src\\data.properties");
            //step3
            prop.load(fis);
            System.out.println(prop.getProperty("browser"));
          }
        }
